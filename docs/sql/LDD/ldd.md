# Langage de Définition des Données

## Syntaxe

```sql
--  Syntaxe
<ORDRE> <TYPE_OBJET> <nom_objet>

--  Les ordres
    CREATE    --  Création de la structure
    ALTER     --  Modification de la structure
    DROP      --  Suppression des données et de la structure

--  Type d’objet
    TABLE - SEQUENCE - VIEW – …
```
