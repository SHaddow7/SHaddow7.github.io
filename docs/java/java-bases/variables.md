## Les variables

### Les types de primitives

| Types   | Taille (octets) |               |
| :------ | :-------------: | :------------ |
| byte    |        1        | -128 à 127    |
| short   |        2        | -32K à 32K    |
| int     |        4        | -2B à 2B      |
| long    |        8        |               |
| float   |        4        |               |
| double  |        8        |               |
| char    |        2        | A, B, C, ...  |
| boolean |        1        | true ou false |

### Déclaration des variables

```java
int num = 5;
float floatNum = 5.99f;
char letter = 'D';
boolean bool = true;
String chaine = "Ceci est une chaîne";
```
